简介

Apache JMeter 是100%纯java语言开发的负载测试和性能测试开源工具。

功能

Apache JMeter可以对静态/动态资源进行性能测试，模拟多个用户并行请求资源端，以测试其强度或分析不同负载情况下的整体性能。

使用

1.下载Apache JMeter

访问http://jmeter.apache.org，点击菜单Download,下载对应程序。

或者使用brew 命令下载

2.运行JMeter

或者 打开Terminal (终端)-> 使用cd 命令进入 apache-Jmeter/bin 目录下运行jemeter.sh文件

3.运行效果

现在可以正式使用JMeter工具
