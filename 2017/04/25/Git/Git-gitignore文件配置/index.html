Git - gitignore文件配置

Git

文章目录

概述

使用gitignore文件来忽略Git跟踪的文件

使用

在Git管理的文件夹中建立.gitignore 文件,在gitignore文件的每一行指定一种过滤规则。Git通常检查gitignore从最高到最低（在一级优先，最后匹配规则决定）

快速建立项目的gitignore文件

gitignore.io － 快速建立项目.gitignore文件

在输入框输入项目相关的操作系统、IDE、编程语言关键字搜索对应的项目gitignore文件,点击Create按钮

例子：VisualStudio 的.gitignore文件
