简述

Jenkins是一个用Java编写的开源的持续集成工具

安装

Linux系统

实战环境

  • Centos 7
  • Java 8

操作

查看官方资料，找到一下命令,这里需要用到wget包没有则

除了这种方式还可以手动下载.war文件进行部署,更多版本可以查看
https://updates.jenkins-ci.org/download/war

Window系统

访问Jenkins官方,下载Window版本的安装包,直接运行msi文件安装

配置

Jenkins安装目录: /var/lib/jenkins/

Jenkins配置文件位置：/etc/sysconfig/jenkins

/etc/sysconfig/jenkins 配置

JENKINS_PORT 是Jenkins的端口，默认端口是8080。

修改后记得重启服务

使用

查看initialAdminPassword文件查询密钥，粘贴到输入框进行验证

Window系统位置：F:\Program Files (x86)\Jenkins\secrets

接着就是插件安装，可以选择自动安装和手动安装,默认就好了。

现在已经完成了Jenkins的安装了
